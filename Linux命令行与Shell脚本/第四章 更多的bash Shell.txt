4.1 检测程序
	
	4.1.1 探查进程
		$ps 
		#ps命令有数不清的参数，这让ps命令成了最难掌握的命令
		
		默认情况下（不带参数），ps命令只会显示运行在当前控制台
	下的当前用户的进程。
		
		Linux系统中使用的GUN ps命令支持三种不同类型的命令行参数
		：
		（1）Unix风格的参数，前面加单破折线
		（2）BSD风格的参数，前面不加破折线
		（3）GUN风格的参数，前面加双破折线
		
		1、仅介绍Unix风格的参数
			-e			显示所有进程
			-f			显示完整格式的输出
			-o format	仅显示由format指定的列
			-L 			显示进程中的线程
			-F 			显示更多额外输出（相对于-f参数）

	4.1.2 实时监测进程
		$top 
		top命令实时显示进程信息，输出的第一部分是系统概况：第
	一行显示系统当前时间、系统的运行时间、登录的用户数以及系
	统的平均负载。
		平均负载有三个值，分别是最近一分钟的、5分钟的、15分钟
	的。值越大系统负载越高。
		第二行显示了进程概要信息——top命令将进程叫作任务（tas
	k）；有多少进程处在运行、休眠、停止或僵尸状态。
		剩下的就是cpu与内存mem的一些相关信息。
		
		最后一部分显示了当前运行中的详细列表，与ps命令类似。
		
	4.1.3 结束进程
		
		1、kill命令
			$kill -l 
			#查看信号
			发信号
			$kill -9 2598	#向进程发送9号信号（SIGKILL）
		
		2、killall 命令
			$killall http*
			#结束了所有以http开头的进程
		
4.2 监测磁盘空间
	
	4.2.1 挂载存储媒体
		Linux文件系统将所有的磁盘都并入一个虚拟目录下。在使用新
	的存储媒体之前，需要把它放入到虚拟目录中，这项工作称作挂载
	。
		1、mount命令
			$mount 
			#mount命令会输出当前系统上挂载的设备列表。
		
		2、umount命令
			从Linux系统移除一个可移动设备时，不能直接从系统移除
		，而应该先卸载。
			
	4.2.2 使用df命令
		$df 
		#df命令可以让你很方便地查看所有已挂载磁盘的使用情况。
		一个常用的参数是-h ，通常用M来代替兆字节，用G来代替吉
	字节。
		$df -h
		
	4.2.3 使用du命令
		$du
		#du命令可以显示某个特定目录（默认当前目录）的磁盘使用
	情况。该方法可以快速判断系统上是不是有超大文件。
		常用参数为：
			-c 显示出所有已列出的文件总大小
			-h 用k，M，G为单位
			-s 显示每个输出参数的总计
	
4.3 处理数据
	Linux系统提供了一些命令行工具来处理大量数据。
	
	4.3.1 排序数据
		$sort file1 
		#sort命令是对数据进行排序的，按照指定的默认语言排序规则
	对文本文件中的数据排序。
		但是默认将数字当成字符处理，所以为了对数字进行排序。可
	以加上参数 -n 把数字识别成数字进行排序，而不是字符。
		另一个常用参数是-M，按月排序，Linux日志文件经常会在每
	行的起始位置有一个时间戳，来表示事件是什么时候发生的。
		-r 参数将数据降序输出。
		还有很多参数就不叙述了。
	
	4.3.2 搜索数据
		grep命令
		格式如下：
		$grep [options] pattern [file]
		grep命令会在输入或指定的文件中查找包含匹配指定模式的字
	符的行。grep的输出就是包含了匹配模式的行。
		
		如果要进行反向搜索（输出不匹配该模式的行），用-v参数
		$grep -v pattern file
		
		如果要显示匹配模式的行所在的行号，加-n参数
		$grep -n pattern file
		
		如果要知道有多少行还有匹配的模式，加-c参数
		$grep -c pattern file
		
		如果要指定多个匹配模式，可以加-e参数
		$grep -e pattern1 -e pattern2 file
		
		默认情况下，grep命令用基本的Unix风格的正则表达式来匹配模
	式。
	
	4.3.3 压缩数据
		Linux下的常用文件压缩工具：bzip2、gzip、zip。
		gzip是Linux上最流行的压缩工具。
		$gzip 
		#压缩文件
		$gzcat
		#查看压缩过的文件内容
		$gunzip
		#用来解压文件
		还支持通配符压缩，压缩所有匹配通配符的文件
		
	4.3.4 归档数据
		Unix和Linux上最广泛使用的归档工具是tar命令。
		tar命令格式如下：
		tar function [options] object1 object2 ...
		
		function 参数都有
		功能	长名称		描述
		-A 		.		将一个已有归档文件追加到另一个已有归档文件
		-c		.		创建一个新的tar归档文件
		-d		.		检查归档文件和文件系统的不同之处
		-r				追加文件到已有归档文件末尾
		-t 				列出已有归档文件的内容
		-u				将归档文件中修改的文件更新追加到归档文件中
		-x				从已有tar归档文件中提取文件
		
		常见选项：
		选项			描述
		-C #dir			切换到指定目录
		-f #file		输出结果到文件或设备file
		-j				将输出结果重定向给bzip2目录来压缩内容
		-p				保留文件权限
		-v				在处理文件时显示文件
		-z 				将输出重定向给gzip命令来压缩内容

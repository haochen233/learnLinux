大多数Linux发行版的默认shell都是GUN bash shell

3.1 启动shell
	GUN bash shell 能提供对Linux系统的交互式访问。
	通常是在用户登录终端时启动
	例如：/etc/passwd 
	其中一行为
	root:x:0:0:root:/root:/bin/bash
	冒号“:”后面的/bin/bash 指定了使用shell程序
	
3.2 shell 提示符
	默认shell提示符是美元符 "$" 这个符号表示等待用户输入。
	
3.3 bash手册
	$man 命令访问存储在Linux系统上的手册页面
	如果不记得命令名，可以使用关键字搜索手册页，
	语法为：
	$man -k 关键字
	
3.4 浏览文件系统
	Linux中使用正斜杠'/'来在文件路径中划分目录
	Windows中使用反斜杠'\'来在文件路径中划分目录
	因为Linux反斜杠来标识转义字符
	
	常见的Linux顶层虚拟目录及其内容
---------------------------------------------------------------
	目录		用途
	/			虚拟目录的根目录，这里一般不会存储文件
	/bin		二进制目录，存放许多用户级GUN工具
	/boot		启动目录，存放启动文件
	/dev		设备目录，Linux在这里创建设备节点
	/etc		系统配置目录
	/home		主目录（家目录）用户目录
	/lib		库目录，存放系统与应用的库文件
	/media		媒体目录，可移动媒体设备的常用挂载点
	/mnt		挂载目录，另一个可移动媒体设备的常用挂载点
	/opt		可选目录，存放第三方软件包和数据文件
	/proc		进程目录，存放硬件和当前进程的信息
	/root		root用户的主目录
	/sbin		系统二进制目录，存放GUN管理员级工具
	/run		运行目录，存放系统运行时数据
	/srv		服务目录，存放本地服务的相关文件
	/sys		系统目录，存放系统硬件信息的文件
	/tmp		临时目录，可以在这创建和删除临时工作文件
	/usr		用户二进制目录大量用户级GUN工具和数据存储在这
	/var		可变目录，用于存放经常变化的文件比如日志文件
	
	3.4.2 遍历目录
			cd destination

3.5 文件个目录列表
		$ls
		$ls -F 区分文件和目录	注意：$l 也可以
		
	3.5.3 过滤输出列表
		$ls -l my_script	
			可以使用模式匹配：
			'?'代表一个字符
			'*'代表0或多个字符
		
			还可以使用中括号'[]'
			中括号表示一个范围
		例如：
			$ls -l file[a-y].txt
			a到y之间的所有字母
		
			另外，还可以使用感叹号'!'，将不需要的内容排除在外
		例如：	
			$ls -l 1[!2]3.txt
			结果中不会有123.txt
	
3.6	处理文件
	3.6.1 创建文件
		$touch fileName
	
	3.6.2 复制文件
		$cp cource destination
		
		单点号'.'很适合cp命令
		如果你需要一个其他目录里面文件拷贝到当前目录，
		使用如下实例：
		$cp /etc/profile .
		$cp /etc/profile ./
		将/etc下的profile文件拷贝到当前目录下。
		
		cp -R的威力强大，递归地复制整个目录的内容
		
		scp命令可以远程复制文件。  
		从本地复制到远程
		# 拷贝文件
		scp /home/test/test.txt root@192.168.0.2:/home/test/
		# 拷贝目录
		scp -r /home/test/ root@192.168.0.2:/home/test/
		
		# 拷贝文件
		scp root@192.168.0.2:/home/test/ /home/test/test.txt
		# 拷贝目录
		scp -r root@192.168.0.2:/home/test/ v/home/test/
		
		
	3.6.4 链接文件		
		软链接：
			类似Windows中的快捷方式。引用数不变。
		硬链接
			原文件的引用，还是原文件（inode号一样），引用数+1
		
				
		不过需要注意的是修改其中任意一个全部都会变化，
		但是如果删除原文件，那么软连接将不可用。
	
	3.6.5 重命名文件
		略。。
		
	3.6.6 删除文件
		养成$rm -i XXX 
		删除时最好加上-i 选项，来提示你

3.7 处理目录
	
	3.7.1 创建目录
		$mkdir dirName
		
		可以根据需要批量的创建目录和子目录
		需要加上选项-p 
		$mkdir -p 1/2/3
	3.7.2 删除目录
		$rmdir dirName
		默认情况下只删除空目录
		rm -rf 递归强制删除	慎用，最好加上-i选项，让系统询问
	
		tree工具是一个工具，可以让你直观的看到目录结构，十分
	美观。
	
3.8 查看文件内容
	3.8.1 查看文件类型
		$file  fileName
		还是非常常用的。
	3.8.3 查看整个文件
		1、cat命令

		2、more
